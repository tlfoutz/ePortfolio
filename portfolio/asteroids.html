Asteroids:
A Retro Remake

Asteroids was a popular video game from the 1980s. The goal of the game is to destroy all the asteroids that appear while navigating the spaceship through the asteroid field without getting hit. To add a layer of complexity, large asteroids break into multiple medium asteroids once hit, and medium asteroids break into multiple small asteroids. With each separation, the asteriods increase in speed.

This program was completed as an assignment for the BYUI CS 165 Object-Oriented Software Development course. While the code for the game framework and GUI interface were supplied, my task was to program and develop classes for the in-game objects that moved. To avoid redundant code, I implemented a parent flyingObjects class with subclasses for the asteroids, ship, and bullets.
