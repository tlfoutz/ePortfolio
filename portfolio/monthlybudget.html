Monthly Budget:
My First Program

A simple program that produces a report for managing personal finances during a month. It takes input from a user, performs calculations, and displays results to the screen.

This was the first practical program I developed as an assignment for my BYUI CS 124 Intro to Software Development course. The main focus was to practice procedural programming techniques, as well as the design process. Before writing any code, I wrote it out in pseudocode and created a structure chart for the functions.
